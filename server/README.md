# Admin - SNS Deepfake

## Requirements

- node: v20.11.1
- yarn: v1.22.19 (don't use npm)
- jest (testing)
- babel (compile & build)

- VSCode Extensions: ESLint, Prettier, REST Client

## Initial setup

- Copy and update file `.env.example` to `.env`
- Install required library packages from scrips
  ```
  yarn sync
  
  OR
  
  yarn install --frozen-lockfile
  ```
